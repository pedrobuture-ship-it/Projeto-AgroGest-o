O AgroGestão é uma plataforma digital que permite o gerenciamento completo de uma fazenda, incluindo controle de rebanho, saúde animal, reprodução, financeiro e geração de notificações automáticas.

O sistema foi projetado para funcionar tanto offline (mobile) quanto online (web), garantindo flexibilidade em ambientes rurais com acesso limitado à internet.

 Funcionalidades

 Dashboard com visão geral da fazenda

 Controle de animais (peso, status, lote, raça)

 Gestão de lotes e pastagens

 Controle de saúde (vacinação e tratamento)

 Reprodução (inseminação, prenhez, partos)

 Registro de pesagens e ganho de peso

 Controle financeiro (receitas e despesas)

 Sistema de notificações automáticas

 Sistema de usuários (dono e funcionários)

   Arquitetura

O sistema foi dividido em duas abordagens:

Offline-first (mobile): permite uso em áreas rurais sem conexão com internet

Cloud (web): sincronização de dados, acesso remoto e gestão centralizada

  Objetivo

Facilitar a gestão de propriedades rurais através da tecnologia, permitindo:

tomada de decisão baseada em dados

aumento de produtividade

melhor controle sanitário e reprodutivo
